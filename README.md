# CIS5500-Final-Project-Group-51

This repository contains the full-stack implementation of **SocioEconAdopt**, a dog adoption recommendation platform that integrates socioeconomic data, adoption records, and user preferences to provide intelligent matching and insights.  
All frontend, backend, and data-processing code required for the project is included in this repository.

## Backup Setup
```bash
cd backend-folder(server.js)
npm install express cors pg
node server.js

## Fronted Setup (open another terminal to do that)
cd client
npm install
npm start
